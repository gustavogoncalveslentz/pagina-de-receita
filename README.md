## 📌 Sobre o Projeto

Este projeto foi construído com foco em **estrutura semântica, estilização refinada e design totalmente responsivo**. A interface foi inspirada nos tons ricos do café e do chocolate, oferecendo uma experiência de leitura limpa e agradável para quem está cozinhando.

O layout adapta-se perfeitamente a dispositivos móveis e desktops, garantindo que os ingredientes e o modo de preparo estejam sempre visíveis e fáceis de acompanhar na cozinha.

<br>

## 🎨 Identidade Visual & Cores

Para refletir a essência da receita, a interface utiliza uma paleta de cores acolhedora e sofisticada:

| Cor | Hex | Aplicação no Projeto |
| :--- | :--- | :--- |
| **Café Escuro** | `#2B1B17` | Textos principais e títulos de destaque |
| **Chocolate** | `#4E3629` | Subtítulos e elementos de transição |
| **Creme Suave** | `#FDFBF7` | Fundo principal para leitura confortável |
| **Toque de Café** | `#8B5A2B` | Detalhes, bordas secundárias e marcadores |

<br>

## 🚀 Tecnologias Utilizadas

O projeto foi desenvolvido utilizando as melhores práticas de desenvolvimento web estático:

- **HTML5:** Estruturação semântica de tags para melhor acessibilidade e SEO (`<main>`, `<section>`, `<ul>`, `<ol>`).
- **CSS3:** Estilização avançada, uso de variáveis internas para consistência de cores, tipografia importada e layouts flexíveis.
- **Responsividade:** Uso estratégico de Media Queries para garantir o visual impecável em qualquer tamanho de tela.
- **GitHub Pages:** Hospedagem automatizada e integrada diretamente com o repositório.
